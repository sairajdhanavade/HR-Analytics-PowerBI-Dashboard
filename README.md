# HR Analytics: Employee Attrition & Retention Dashboard (Power BI)

## 📌 Project Overview
This project leverages **Power BI** to analyze HR data (IBM Attrition Dataset) and identify key factors influencing employee turnover. The dashboard provides a comprehensive view of workforce demographics, satisfaction levels, and attrition trends to help HR departments make informed retention decisions.

## 🛠️ Data Engineering (Power Query)
In this project, I performed the following data transformation steps:
*   **Data Cleaning:** Handled missing values and removed redundant columns like `Over18` and `EmployeeCount`.
*   **Conditional Columns:** Created age groups (e.g., 18-25, 26-35) and salary brackets for better segmentation.
*   **Data Modeling:** Established relationships between tables and ensured a clean Star Schema for optimal performance.

## 📊 Key Dashboard Features
*   **KPI Cards:** Total Employees, Attrition Count, Attrition Rate, and Average Age.
*   **Attrition by Department:** Visualizing which departments (Sales, R&D, HR) face the highest turnover.
*   **Job Satisfaction Heatmap:** Correlation between Job Role and Satisfaction ratings.
*   **Income vs. Attrition:** Analyzing if lower-income brackets correlate with higher exit rates.
*   **Slicers:** Interactive filters for Department, Job Role, and Gender to allow deep-dive analysis.

## 💡 Key Insights & Findings
*   **Critical Attrition:** Sales Representatives have the highest attrition rate at over 30%.
*   **Work-Life Balance:** Employees rating '1' (Bad) for Work-Life Balance are 2x more likely to leave.
*   **Travel Factor:** Frequent travelers show a significantly higher tendency to attrite compared to non-travelers.
